📘 Documentazione – TechFlow Pro

Guida rapida per utilizzare e personalizzare il template

📁 Struttura dei file

  • index.html – Landing page principale
  • about.html – Pagina "Chi siamo"
  • contact.html – Form di contatto
  • components.html – Libreria componenti
  • README.md, LICENSE.txt, docs.html

🖊️ Come modificare i contenuti

Apri i file con un editor di testo (es. VSCode). Puoi modificare facilmente:

  • Testi: cerca il testo all'interno dei tag <h1>, <p>, ecc.
  • Immagini: sostituisci i link src="images/..." con i tuoi file
  • Prezzi: modificabili nella sezione #pricing di index.html

✉️ Newsletter

Nel file index.html, cerca il form con action="YOUR_MAILCHIMP_URL" e sostituiscilo con il link del tuo form Mailchimp.

💻 Componenti riutilizzabili

Apri components.html per visualizzare pulsanti, card, modali, input, ecc. Ogni blocco è pronto per il copia/incolla e include un pulsante “📋 Copia codice”.

📐 Personalizzazione design

Puoi cambiare i colori, padding, dimensioni modificando le classi Tailwind nei tag HTML. Esempio:

<button class="bg-blue-600 hover:bg-blue-700 px-4 py-2">Cambia colore</button>
